triangulate images

drop an image in the browser to triangulate it. this script uses the delaunay triangulation algorithm. it is based on the triangulation image generator and includes some speed improvements. this experiment was created by georg. you can follow him on twitter or explore the source code on github.

you can check out some of his other javascript experiments like jpg-glitch, and distort-grid on github, too.
